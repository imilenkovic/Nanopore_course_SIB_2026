Introduction to dRNA sequencing and correponding analyses

# Workshop Timetable

## Day 1

| Time          | Session                                                                                     |
|----------------|---------------------------------------------------------------------------------------------|
| 09:00–10:00   | **Introduction**                                                                            |
| 10:00–10:30   | **Break**                                                                                   |
| 10:30–11:00   | **Demultiplexing, basecalling, mapping**                                                    |
| 11:00–12:00   | **Hands-on: Server login, basecalling, mapping**                                            | Practice pipeline setup, basecalling, and mapping workflows. |
| 12:00–13:00   | **Lunch Break**                                                                             | — |
| 13:00–14:30   | **Indirect RNA modification mapping in ribosomal RNA**                                      | Learn principles of non-random errors in RNA modification detection. Hands-on: basecall with agnostic models and extract modification info with [EpiNano](https://github.com/enovoa/EpiNano). |
| 14:30–15:00   | **Break**                                                                                   | — |
| 15:00–17:00   | **Direct RNA modification mapping in mRNA**                                                 | Explore principles of RNA modification basecalling. Hands-on: basecall with modification-aware models and extract data with [Modkit](https://github.com/nanoporetech/modkit). |
